# Kubectl Reference

This tool creates a DocBook documentation from the Kubectl inline help.

```
# Create the Docbook file
$ make docbook

# Create a PDF file, in USletter format
$ make pdf

# Create a PDF file, in A4 format
$ make pdf FORMAT=A4
```
